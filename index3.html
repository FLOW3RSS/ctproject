🦫 카피바라 챌린지: AI 강수확률 기반 우산 예측기

고1 수학(함수, 데이터 해석) + 고2 통계·CT(문제분해/추상화)

지문:

이 예측기는 구름량(C: cloud cover), 습도(H), 풍속(W), 기온(T)을 기반으로
나의 가중치와 AI 회귀모델이 각각 우산 휴대 필요성(필수/권장/선택)을 예측합니다.
예측 공식: P = w₁×C + w₂×H + w₃×W + w₄×T (여기서 P=강수확률, w₁~w₄의 합은 1)
(P≥70: 필수, 50≤P<70: 권장, P<50: 선택)

활동 규칙:
  • 지도 클릭 또는 위도·경도 입력 → 위치 지정
  • 날짜 선택 (오늘/5일 중 하나, 달력으로 지정 가능)
  • 가중치(0~1, 합 1) 입력 (구름량/습도/풍속/기온)
  • 예측 결과 : 내 가중치/AI모델/실제 강수확률 모두 그래프와 표로 비교
  • 문제풀이: 둘 중 어느 쪽 예측이 더 실제와 가까웠는지, 일치여부 분석 문항 포함
  • AI 모델 설명: 최근 20일 실제 데이터를 바탕으로 다중회귀(머신러닝)를 수행, 예측값과 실제값의 평균 오차까지 자동 보정해 실제와 매우 유사하게 만듭니다.
